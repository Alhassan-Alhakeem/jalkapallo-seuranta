# Joukkueen otteluseuranta

Selainpohjainen sovellus harrastejalkapalloilijalle joukkueen ottelutulosten ja maalintekijätilastojen seurantaan.

## Käyttö

1. Avaa index.html selaimessa
2. Siirry "Kirjaa ottelu" -välilehdelle
3. Täytä ottelun tiedot ja tallenna

## Ominaisuudet

- Otteluiden lisäys, muokkaus ja poisto
- Kauden yhteenveto (voitot, tasapelit, häviöt, maaliero)
- Maalintekijätilasto

## Tekninen toteutus

- HTML, CSS, JavaScript — ei ulkoisia kirjastoja
- Tiedot tallennetaan selaimen local storageen
